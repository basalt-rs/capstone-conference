What is Basalt?

Basalt is a local, free, and open-source suite for creating, managing, and competing in programming competitions.

Why Basalt?

How do I use Basalt?

  • Setup a router that is not connected to the internet
  • Start the server on a device with a known IP
  • Connect using the web client or desktop application

Components

  • A cross-platform application for web and desktop
  • A highly performant server
  • A command line interface (CLI) utility
  • An expressive and flexible configuration format
  • A process-isolation library
  • A test-runner
Supporting Linux + MacOS + Windows + Web.

Technologies

  • Rust
  • React
  • Tauri
  • Docker
    • We chose Rust for all of our business logic, configuration, and utilities because what you see is what you get. We chose to use Docker because our server is best composed of layers of images.

Running Tests

2 Crates

Leucite

A rust crate for creating a sandbox around any command using Linux's landlock and prlimit features.

Leucite is a published crate that can be used in any context, not just Basalt.

In fact, we've already had open source contributions from a PhD student in Denmark.

Erudite

A fully asynchronous test runner that runs an entire test suite in parallel.

Optionally compiles an application and then runs it against a suite of tests using Leucite to sandbox each test.

Erudite Features

  • Sandbox files
  • Restrict network access
  • Set timeout
  • Restrict memory usage
  • Restrict files created
