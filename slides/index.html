What is Basalt?

Basalt is a local, free, and open-source, ...
Basalt is a local, free, and open-source suite for creating, managing, and competing in programming competitions.

Components

  • A cross-platform application for web and desktop
  • A highly performant server
  • A command line interface (CLI) utility
  • An expressive and flexible configuration format
  • A process-isolation library
  • A test-runner

Technologies

  • Rust
  • React
  • Tauri
  • Docker

Running Tests

2 Crates

Leucite

A rust crate for creating a sandbox around any command using Linux's landlock and prlimit features.

Leucite published crate that can be used in any context, not just Basalt.

In fact, we've already had open source contributions from a PhD student in Denmark.

Erudite

A fully asynchronous test runner that runs an entire test suite in parallel.

Optionally compiles an application and then runs it against a suite of tests using Leucite to sandbox each test.
